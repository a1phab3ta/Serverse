Help: Report Generation

Step 1: Navigate to the Report Page

Click the "Generate Report" button at the top of your main screen, right below the search bar.

Step 2: View Report

Once on the report page, you can see all of the changes that have been made to the partners, like deletion, addition, or edits. You can also search among the changes using the search bar.

Step 3: View History

Click on any Partner Name in the file to view the state of the partner when the change was made.

Step 4: History

You can see all the data about the partner before it was changed here.
